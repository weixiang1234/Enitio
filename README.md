# ESP32 PlatformIO Project

This project is written in C using PlatformIO in Visual Studio Code.

## Hardware Used

- ESP32 Dev Board
- PCA9685 Servo Driver
- Joystick Module
- Servo Motors

## Project Description

This project controls servos using joystick input.

## How to Build

Open the project in VS Code with PlatformIO installed, then click:

PlatformIO > Build

## How to Upload

Connect the ESP32 board and click:

PlatformIO > Upload
